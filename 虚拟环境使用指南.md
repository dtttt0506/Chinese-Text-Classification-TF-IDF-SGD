# 数据挖掘项目虚拟环境使用指南

## 虚拟环境创建和激活

### 创建虚拟环境
```bash
python -m venv venv
```

### 激活虚拟环境
**Windows:**
```bash
.\venv\Scripts\activate
```

**Linux/Mac:**
```bash
source venv/bin/activate
```

### 停用虚拟环境
```bash
deactivate
```

## 依赖包安装

### 安装所有依赖包
```bash
pip install -r requirements.txt
```

### 安装单个包
```bash
pip install 包名
```

### 保存当前环境依赖
```bash
pip freeze > requirements.txt
```

## 常用命令

### 查看已安装包
```bash
pip list
```

### 查看包信息
```bash
pip show 包名
```

### 升级包
```bash
pip install --upgrade 包名
```

### 卸载包
```bash
pip uninstall 包名
```

## 项目结构
```
project_root/
├── venv/                    # 虚拟环境目录
├── requirements.txt         # 项目依赖
├── run.py                   # 项目入口
├── config/
│   └── config.yaml         # 配置文件
├── src/                    # 源代码
├── data/                   # 数据文件
├── output/                 # 输出结果
├── checkpoints/            # 模型保存
└── logs/                   # 日志文件
```

## 快速开始

1. 激活虚拟环境
2. 安装依赖包
3. 运行项目：
```bash
python run.py
```

## 注意事项

- 始终在虚拟环境中运行项目
- 定期更新requirements.txt文件
- 不同项目使用不同虚拟环境，避免包冲突
